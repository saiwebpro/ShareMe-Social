== Description ==
ShareMe Social is a customizable social media sharing plugin for WordPress. Users can enable or disable icons for Facebook, Twitter, and LinkedIn, set custom colors, and position the icons above or below the content.

== Installation ==
1. Upload the `shareme-social` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings > ShareMe Social to configure the plugin options.

== Frequently Asked Questions ==
= Can I change the icon color? =
Yes, you can select the icon color in the settings.
